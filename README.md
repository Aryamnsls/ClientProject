# Knight Errant Technologies Website

A modern and responsive business website developed for Knight Errant Technologies. The website is designed to showcase company services, strengthen online presence, and provide a seamless user experience across desktop and mobile devices.

## 🚀 Features

* Modern and responsive UI
* Professional company branding
* Mobile-friendly design
* Fast page loading
* Service showcase section
* About Us page
* Contact page
* Search functionality
* SEO-friendly structure

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* Vite
* Git & GitHub

## 📂 Project Structure

```bash
src/
├── components/
├── pages/
├── assets/
├── styles/
public/
package.json
```

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Aryamnsls/ClientProject.git
```

Navigate to the project folder:

```bash
cd ClientProject
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## 📦 Production Build

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
